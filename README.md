# SIJARU

Sistem Informasi Jalan Rusak berbasis Laravel.

## Tentang SIJARU

SIJARU adalah aplikasi pelaporan jalan rusak berbasis web yang memudahkan masyarakat melaporkan kerusakan jalan secara cepat, akurat, dan berbasis lokasi.

## Fitur

- Login Multi User
- Data Laporan Jalan Rusak
- Data Masyarakat
- Data Admin
- Peta Jalan Rusak
- Export PDF
- Export Excel
- Email Notifikasi Status Laporan

## Teknologi

- Laravel 12
- PHP 8.2
- MySQL
- Bootstrap
- AdminLTE
- Leaflet Maps
